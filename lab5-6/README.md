## Техническое задание: Проект "Система управления личной библиотекой"

Выполнил: Ручкин Иван СКБ251

Консольное приложение для управления личной библиотекой с использованием объектно-ориентированного программирования.

Цель:
Написать консольную программу, которая:
- выводит информацию о пользователе и книгах
- позволяет пользователю получать и возвращать книги
- сохраняет, загружает и парсит данные из файла
- позволяет добовлять книги и пользователей
- Поиск книги по ISBN
- валидирует входные данные

##### Требования (C++17, CMake 3.10+)

### Сборка

``` bash
git clone https://github.com/kernel-paniccc/MIEM_C_plus_plus.git

cd MIEM_C_PLUS_PLUS/lab5-6

cmake .
make
./library
```
